## 🦆 Duck Race

**Set the Ducks, Start the Timer, and Let Fate Decide the Winner!**

**Duck Race** is a fun little party-style mini-game built using **Cocos** where everything is about luck and entertainment. It’s perfect for random giveaways, decision-making games, or just casual fun with friends.

---

### 🎮 How to Play

- ➕ **Setup**: Choose how many ducks to race and give them custom names.
- ⏱️ **Time It**: Set the duration of the race countdown.
- ▶️ **Start**: Hit the "Start" button and wait for the timer to count down.
- 🏁 **Result**: Once time's up, one duck is randomly chosen as the loser (or winner — it’s up to how you define it)!

---


### 📝 Overview

Duck Race focuses on **random outcome + visual fun**. It’s great for:
- 🎁 Giveaway tools (random picker)
- 🧠 Quick party mini-game
- 😂 Fun punishment selector

Key Features:
- 🔢 Configurable number of ducks
- ✍️ Custom duck names
- ⌚ Settable race duration
- 🎲 Fair and simple randomization logic
- 💻 Built using Cocos Engine
