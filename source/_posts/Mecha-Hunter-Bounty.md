## 🤖 Mecha Hunter: Bounty — Post-Apocalyptic TPS Action

**Lock. Load. Ride.**  
Set in a post-apocalyptic wasteland, **Mecha Hunter: Bounty** is a third-person shooter that fuses gunplay, tactical vehicle handling, and randomized missions. Play as **Ace Cazador**, a mechanical bounty hunter tasked with eliminating rogue AI robots in a shattered world.

---

### 📝 Overview

- 🎮 **Genre**: Third-Person Shooter + Vehicle Combat
- 🧭 **Theme**: Post-apocalyptic western with machine bounty hunters
- 🕹 **Platform**: Android / iOS
- 🧱 **Graphics**: 3D Low-Poly Stylized
- 🔧 **Engine**: Unity 3D
- 👤 **Mode**: Single Player

Design focuses on **fast-paced combat**, **mobile-friendly controls**, and **engaging repeatable gameplay**.

---

### 🧠 World & Lore

The western lands are no longer safe — rogue AI have turned the desert into a battlefield.  
You are **Ace Cazador**, a bounty hunter equipped with top-tier weaponry and customizable vehicles. Complete contracts, reclaim lost tech, and survive extraction — all while fending off intelligent machines.

---

### 🔫 Core Features

#### 🎯 Dynamic Combat

- Equip 2 weapons per mission with unique traits
- Features **laser-sight** and **auto-lock** for precise mobile targeting
- Dual joystick controls:
    - Left joystick: movement
    - Right joystick: camera + aiming

---

#### 🤖 Diverse Enemies

| Type           | Behavior                              |
|----------------|---------------------------------------|
| Melee Bot      | Rushes and attacks in close combat    |
| Ranged Bot     | Fires from distance and evades shots  |
| Boss: Flamethrower | Massive robot with AoE flame attacks |
| Boss: Warhammer   | Slams ground, causes shockwaves     |

---

#### 🧨 Mission Types (5)

| Mission         | Objective                          |
|-----------------|-------------------------------------|
| Elimination     | Destroy specific targets            |
| Core Recovery   | Retrieve stolen energy cores        |
| Base Defense    | Hold position and survive waves     |
| Vehicle Retrieval | Recover stolen vehicles           |
| Timed Escape    | Reach evac point before time runs out |

---

#### 🚗 Realistic Vehicle Control

- Supports FWD / RWD / AWD mechanics
- Perform **drifts**, **ram attacks**, or **eject before explosion**
- Touch controls: throttle, brake, steer, enter/exit vehicle

---

#### 🧭 Randomized Levels

- Each mission generates a unique layout
- Built-in **mini-map** (top-right) for orientation and objectives

---

### 📱 Mobile Control Scheme

| Action             | Control                                |
|--------------------|----------------------------------------|
| Move               | Left Joystick                          |
| Camera / Aim       | Right Joystick + Lock-On               |
| Fire               | Large fire button                      |
| Reload             | Reload button                          |
| Weapon Swap        | Icon under HUD                         |
| Enter/Exit Vehicle | Button near minimap                    |
| Interact           | Shows near items (boxes, cars, etc.)   |

---

### 🎯 HUD Layout

| Position         | UI Element                             |
|------------------|-----------------------------------------|
| Top Left         | HP bar                                  |
| Top Right        | Weapon icons, ammo count                |
| Center           | Mission objectives                      |
| Top Left Corner  | Mini-map + toggle button                |

---

### 🧨 Weapon Arsenal

| Weapon           | Description                                                                 |
|------------------|------------------------------------------------------------------------------|
| Automatic Pistol | Fast rate, high accuracy, ideal while moving                                |
| Cowboy Revolver  | High damage, short range, great for critical hits                           |
| Assault Rifle    | Balanced power and precision                                                |
| Shotgun          | Strong spread, great for clearing close groups                              |
| Sniper Rifle     | Long-range, armor-piercing, one-shot potential                              |
| Minigun          | High RPM, great for crowd control, hard to aim                              |
